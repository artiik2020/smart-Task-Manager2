# Smart-Task-Manager RU
'умный список задач' позволит вам не волноваться о том что вы что то забыли или что бы вас вовремя предупредили
о чём то что вы хотите. Приложение позволяет вам импортировать ваш список задач или же изменить стиль приложения,
а так же у вас будет локализация так что не бойтесь что вы что то не поймёте

# Smart-Task-Manager ТЗ
составление ТЗ: Проект: "Умный" Список Задач/ Smart Task Manager
* Базовый функционал:
  *   Добавление категорий и задач
  *   удаление категорий и задач
  *   редактирование задач -Отметка о выполнении -Сохранение списка в файл (QSettings или JSON через QJsonDocument)
  *   возможность импорта категорий и задач
  *   возможность менять вид окна с помощью css
* Дополнительные функции:
  * Системное уведомление
  * Категории
  * Теги
  * Локализация (EN/RU)
  * удобное пользование функциями для любого пользователя

    
# Smart-Task-Manager EN
'smart task list' will allow you not to worry about what you have forgotten or that you would be warned in time
about something that you want. The application allows you to import your task list or change the style of the application,
and also you will have localization so don't be afraid that you don't understand something

# Smart-Task-Manager TK
compilation of TK: Project: "Smart" Task List/ Smart Task Manager
* Basic functionality:
 * Adding categories and tasks
 * deleting categories and tasks
 * editing tasks -Marking tasks as completed -Saving the list to a file (QSettings or JSON via QJsonDocument)
 * importing categories and tasks
 * changing the window's appearance using CSS
 * Additional features:
 * System notifications
 * Categories
 * Tags
 * Localization (EN/RU)
 * User-friendly features for all users
